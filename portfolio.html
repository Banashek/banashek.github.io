Ant MBA

Source

A game about an ant trying to make his name in the business world created in C# with Unity3d in 48 hours for the Ludum Dare #30.

MacroMeal

Source

A website for users to create and upload recipes with macronutrient content to help alleviate the issue of finding recipes that match their macronutrient requirements. Built as a learning project with Ruby on Rails in a weekend. Utilizes Devise for user authentication with custom built authorization utilizing a Postgresql database hosted on Heroku.

Me, You, & Hue

Source

Me, You, & Hue, a.k.a. Project ColorDare is a 3-player co-op platform-puzzler made in C# with Unity3d in under 72 hours for Ludum Dare #29.

Flappy_FPS
